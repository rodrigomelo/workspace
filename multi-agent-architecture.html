🐚 Agentes no Raspberry Pi

O Raspberry Pi executa agentes OpenClaw via SSH. Para executar comandos:

ssh -i ~/.ssh/id_ed25519 -o StrictHostKeyChecking=no melor@192.168.15.17 "comando"

Exemplos de Comandos

  • ssh -i ~/.ssh/id_ed25519 melor@192.168.15.17 "hostname" - Verificar conexão
  • ssh -i ~/.ssh/id_ed25519 melor@192.168.15.17 "uptime" - Tempo de atividade
  • ssh -i ~/.ssh/id_ed25519 melor@192.168.15.17 "docker ps" - Containers ativos
  • ssh -i ~/.ssh/id_ed25519 melor@192.168.15.17 "htop" - Monitoramento

🔑 Configuração SSH

Adicionar ao ~/.ssh/config para简化:

Host pi
    HostName 192.168.15.17
    User melor
    IdentityFile ~/.ssh/id_ed25519
    StrictHostKeyChecking no

Depois use: ssh pi "comando"
