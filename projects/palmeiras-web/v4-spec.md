# Palmeiras Dashboard v4 - Design Specification

## Overview
- **Version:** 4.0
- **Theme:** Lighter, cleaner design
- **Goal:** Better readability, more whitespace

## Color Palette
```
--bg-light: #F5F5F5
--bg-white: #FFFFFF
--palmeiras-green: #007824
--text-primary: #1A1A1A
--text-secondary: #666666
--accent: #FFD700 (gold)
```

## Layout

### Header
- White background
- Palmeiras logo/title in green
- Clean, minimal

### Cards
- White background with subtle shadow
- Rounded corners (12px)
- Green accent borders
- More padding/whitespace

### Sections
1. **Próximo Jogo** - Hero card
2. **Agenda** - Next 5 matches
3. **Resultados** - Last 5 games
4. **Classificação** - Standings
5. **Onde Assistir** - TV/Streaming
6. **Estádio** - Stadium info
7. **Notícias** - News from ge.globo

## Mobile
- Touch-friendly buttons (min 44px)
- Responsive layout
- Horizontal scroll where needed

## Typography
- Headings: Bold, green
- Body: Clean, readable
- Good contrast
