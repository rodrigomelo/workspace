# Project Documentation

Documentação do projeto: ADRs, contexto, referências.

## ADRs (Architecture Decision Records)

Local: `knowledge/decisions/` (compartilhado) ou `docs/adrs/` (projeto-específico)

Formato: YYYY-MMDD-titulo.md

Template:
```
# Título

Status: Proposed | Accepted | Deprecated

Context:
- Por que essa decisão é necessária?

Decision:
- O que decidimos e por quê?

Consequences:
- Prós
- Contras
- Impacto em outros agentes
```

- Dependências externas
