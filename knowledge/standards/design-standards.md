# Design Standards

Padrões visuais e de experiência que todos os projetos seguem.

## Tokens de Design

### Cores
- Primária: `#3B82F6` (azul)
- Secundária: `#10B981` (verde)
- Acento: `#F59E0B` (amber)
- Fundo: `#FFFFFF` (branco) / `#1F2937` (dark)
- Texto: `#111827` (quase preto) / `#F9FAFB` (claro)

### Tipografia
- Família: Inter (UI), JetBrains Mono (code)
- Tamanhos base: 12, 14, 16, 20, 24, 32, 40, 56 px
- Line-height: 1.5 para texto, 1.25 para títulos

### Espaçamento
- Unidade base: 4px (0.25rem)
- Grid: 8px multiples

## Componentes Comuns

- Botões (primary, secondary, icon-only)
- Inputs (text, select, checkbox, radio)
- Cards
- Modais
- Tables
- Alerts / Toasts

Todos devem ser reusáveis e acessíveis (WCAG 2.1 AA).
